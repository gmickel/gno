## Quick Start

```bash
# Install
bun install -g gno

# Initialize with your notes folder
gno init ~/notes --name notes

# Index documents
gno update

# Search
gno search "project deadlines"
gno query "authentication best practices"
gno ask "summarize the API discussion" --answer
```

Ready to dive in? Check out the [Quick Start Guide](/docs/QUICKSTART/) for the full walkthrough.
