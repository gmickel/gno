Getting Started

Up and Running in Minutes

1

Install GNO

Install globally via Bun or npm:

Terminal 
bun install -g gno
2

Add a Collection

Index a folder of documents:

Terminal 
gno add ~/Documents/notes --collection notes
3

Search Your Knowledge

Find what you need with natural language:

Terminal 
gno search "authentication flow diagram"

# Or get a grounded answer:
gno ask "How does the auth system work?" --answer
4

Use with AI Assistants

Start the MCP server for Claude integration:

~/.config/claude/claude_desktop_config.json 
{
  "mcpServers": {
    "gno": {
      "command": "gno",
      "args": ["mcp"]
    }
  }
}
